# 🧮 Calculadora interactiva en Python

 Este es un script de consoloa que permite realizar operaciones matemáticas básicas y almacenar un historial entre sesiones.

## Lógica de Operaciones

- Función `limpiar_pantalla()`: Nos ayuda a mantener un menú limpio y libre de residuos de otras sesiones.

- Función `operaciones()`: Uno de los pilares del código, esta función se encarga de recolectar datos para realizar las operaciones seleccionadas.

- Función `historial()`: Se encarga de crear el historial de operaciones hechas en la función anterior `operaciones()` de manera ordenada.

## Operaciones Soportadas

| Símbolo | Operación | Notas |
| :--- | :--- | :--- |
| `+` | Suma | Soporta decimales |
| `-` | Resta | Soporta decimales |
| `/` | División | Incluye validación contra división por cero |
| `*` | Multiplicación | Soporta decimales |

## Requisitos y Ejecución

### Requisitos
- Python 3
- Sistema operativo compatible con comandos de terminal (`clear` para Unix/Linux/Mac).

### Ejecución
- Copia el link del repositorio y copialo con el comando `git clone <URL del repositorio>`


- Ejecuta en tu terminal: 

``` bash
python main.py 
```
